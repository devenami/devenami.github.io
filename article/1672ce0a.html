docker搭建redis集群

发表于 2021-04-24 分类于 数据库 ， Redis

redis集群提供了一种分布式的redis高可用方案，用于替换老旧的主从加哨兵的模式。

配置redis集群需要如下几步：

  1. 下载redis镜像
  2. 配置网络
  3. 准备redis配置文件
  4. 启动redis
  5. 链接其中一个redis进行创建集群

  3. docker logs 查看指定容器的输出日志
  4. docker restart 重启容器
